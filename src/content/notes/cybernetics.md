# Communication, feedback, agency

Cybernetics studies regulation, communication, feedback, and control in systems.

```text
communication + feedback → continued agency
```

Within Q.U.I.L.L., the operator is not an external user issuing isolated commands. The operator and machine form a coupled system. State is observed, intentions are translated, actions are taken, consequences return as feedback, and the system adapts.

The design objective is not maximal automation. It is sustained agency under changing cognitive, social, material, and affective conditions.
